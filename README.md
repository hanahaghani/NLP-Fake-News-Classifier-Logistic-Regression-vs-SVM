# Fake News Detection using Machine Learning

این پروژه بخشی از مسیر یادگیری من در حوزه Machine Learning و NLP هست که همزمان با مطالعه کتاب Hands-On Machine Learning انجام شده.

هدفم این بود که فقط مفاهیم رو حفظ نکنم، بلکه ببینم چطور میشه اون‌ها رو روی یک مسئله واقعی کنار هم قرار داد.



# 🎯 هدف پروژه

تشخیص اخبار واقعی و جعلی با استفاده از روش‌های کلاسیک NLP و مدل‌های یادگیری ماشین.



# 🧠 مراحل انجام کار

در این پروژه سعی کردم یک Pipeline کامل برای حل مسئله داشته باشم:

- بررسی و تمیز کردن داده‌ها
- ترکیب عنوان و متن خبر برای ساخت ورودی بهتر
- تبدیل متن به بردارهای عددی با TF-IDF
- بررسی داده‌ها در فضای دوبعدی با PCA / t-SNE / LLE
- آموزش مدل‌های مختلف:
  - Logistic Regression
  - LinearSVC
- تنظیم پارامترها با GridSearchCV
- ارزیابی مدل با Cross Validation و Confusion Matrix



# 📊 نتایج

در مقایسه مدل‌ها، LinearSVC عملکرد بهتری نسبت به Logistic Regression داشت.

همچنین بررسی خطاها نشان داد که مدل در بعضی نمونه‌های مرزی هنوز اشتباه دارد که می‌تواند در پروژه‌های بعدی با روش‌های پیشرفته‌تر NLP بهتر شود.


# 💡 نکته مهم برای من در این پروژه

برای من این پروژه فقط درباره گرفتن accuracy نبود.

بیشتر سعی کردم بفهمم:
- چرا TF-IDF جواب می‌دهد
- چرا بعضی مدل‌ها بهتر عمل می‌کنند
- و چرا تغییرات کوچک در preprocessing نتیجه را تغییر می‌دهد
